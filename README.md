## Getting Started

First, run the development server:

1. Up docker database
```bash
docker compose up -d
# or
```
2. Rename `.env.exmaple` to `.env`
3. Replace environment variables

4. Prisma commands
```bash

 npx prisma init
 npx prisma migrate dev
 npx prisma generate

```

* . Run the server
```bash
npm run dev
# or
```
